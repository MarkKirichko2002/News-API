Новостное приложение с различными категориями новостей и поиском новостей. 

Приложение работает с API - NewsCatcher News API. 

В данном проекте присутствует: 
- Микрофон  
- Распознание речи.
 
Стек технологий:
- SwiftUI
- SDWebImageSwiftUI
- Alamofire
- Foundation
- AVFoundation
- Speech
- HidableTabView

В проекте используется архитектурный паттерн MVVM.
